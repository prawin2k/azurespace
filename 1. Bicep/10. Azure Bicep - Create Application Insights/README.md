# Azure Space - 10. Azure Bicep - Create Application Insights

## What is Application Insights?

Application Insights is a minitoring solution that captures Telemetry data of your application and provides insights.

## Execute & Deploy resources to Azure Resource Group using the below command.

```
az deployment group create -g azbicep-dev-eus-rg1 -f 2.AppServicePlan.bicep
```
